# Deployment

### Deployment - Imperative Command
- Both ```deploy``` and ```deployment``` work the same.
```yaml
kubectl create deploy <name-of-deployment> --image=<name-of-image>
kubectl create deployment nginx-deployment --image=nginx
```
**Deployment with Multiple Replicas and/or Namespace**
```yaml
kubectl create deployment nginx-deployment --image=nginx --replicas=3 --namespace=production
```

### Deployment Configuration - Declarative Command 
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
  labels:
    app: nginx
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - image: nginx
        name: nginx
        ports:
        - containerPort: 80
```

### Deployment Template Generation
```yaml
kubectl create deployment <name-of-deployment> --image=<name-of-image> --dry-run=client -o yaml
kubectl create deploy nginx --image=nginx --dry-run=client -o yaml
```

--- 

### Scale the Deployment
```yaml
kubectl scale deployment nginx-deployment --replicas=5
```

### Update the Deployment Container Image
```yaml
kubectl set image deploy <name-of-deployment> <name-of-container-inside-pod>=<name-of-updated-container>
kubectl set image deploy nginx-deployment nginx=nginx:1.28
kubectl set image deployment/nginx-deployment nginx=nginx:1.28
```

---
### Checking Rollout Status
```yaml
kubectl rollout status deployment nginx-deployment
kubectl rollout history deployment/nginx-deployment
```

### Roll Back the Deployment (Immediate)
```yaml
kubectl rollout undo deployment/nginx-deployment
```
**Roll back to a specific revision:**
```yaml
kubectl rollout undo deployment/nginx-deployment --to-revision=2
```
### Restart the Deployment
```yaml
kubectl rollout restart deployment/nginx-deployment
```

---

### Expose the Deployment

**ClusterIP service:**
```yaml
kubectl expose deployment nginx-deployment --name=nginx-service --type=ClusterIP --port=80 --target-port=80
```

**NodePort service:**
```yaml
kubectl expose deployment nginx-deployment --name=nginx-service --type=NodePort --port=80 --target-port=80
```
---

### Delete the Deployment
```yaml
kubectl delete deployment nginx-deployment
```
**Delete Associated Services** (if Deployment is Exposed)  
```yaml
kubectl delete service nginx-service
```
